# NeoVim configuration

Here is my NeoVim config 

Config use LazyLoader for plugins

## Installation on Linux/MacOS

* Save your config

    ```bash
    mv ~/.config/nvim{,.bak}
    ```

* Clone config
    
    ```bash
    git clone https://github.com/farneser/nvim-config ~/.config/nvim
    ```

* Remove `.git` folder

    ```bash
    rm -rf ~/.config/nvim/.git
    ```
